# مقدمة في الشبكات العصبية: كيف تفكر الآلة؟

هل تساءلت يوماً كيف يستطيع هاتفك التعرف على وجهك؟ أو كيف يترجم Google النصوص بين اللغات؟ السر يكمن في **الشبكات العصبية الاصطناعية**.

في هذا المقال، سنأخذك في رحلة مبسطة لفهم أحد أعظم الاختراعات التقنية في تاريخ البشرية.

## ما هي الشبكات العصبية؟

الشبكات العصبية هي نماذج حاسوبية مستوحاة من **هيكلية الدماغ البشري**. تتكون من:

- **الخلايا العصبية (Neurons)**: وحدات معالجة صغيرة
- **الروابط (Connections)**: تمرر المعلومات بين الخلايا
- **الأوزان (Weights)**: تحدد قوة كل رابط

```python
import numpy as np

class SimpleNeuron:
    def __init__(self, input_size):
        self.weights = np.random.randn(input_size)
        self.bias = np.random.randn(1)
    
    def activate(self, x):
        return np.dot(x, self.weights) + self.bias
```

## كيف تعمل الخلية العصبية؟

تخيل أن الخلية العصبية مثل **مصفاة ذكية**:

1. تستقبل مدخلات متعددة
2. تضرب كل مدخل في وزنه
3. تجمع النتائج
4. تمررها عبر دالة تنشيط

### دالة Sigmoid

```python
def sigmoid(x):
    return 1 / (1 + np.exp(-x))
```

هذه الدالة تحول أي رقم إلى قيمة بين 0 و 1، مثالية للاحتمالات!
